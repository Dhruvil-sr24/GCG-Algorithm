## Motivation

Modern jailbreak attacks are increasingly important for evaluating LLM robustness.

This repository aims to provide a clean and modular implementation that can be extended for:

- adversarial robustness research
- AI safety evaluation
- jailbreak benchmarking
- defense development
--- 
Future work includes studying optimization trajectories,
token replacement dynamics, and transferability across different model families.
